# Plugging into the Future
### An Exploration of Electricity Consumption Patterns (India, 2019–2020)

## About the Project
This project analyses daily electricity consumption data across 33 Indian states and 5 regions from January 2019 to December 2020. The study focuses on understanding how the COVID-19 pandemic and nationwide lockdown affected India's energy demand — and how different states recovered afterwards.

## Dataset
•	Source: POSOCO (Power System Operation Corporation of India)
•	File: Consumption.csv
•	Records: 16,600 rows
•	States: 33 States and Union Territories
•	Regions: NR, WR, SR, ER, NER
•	Period: 2019-01-02 to 2020-12-05
•	Fields: States, Regions, Latitude, Longitude, Dates, Usage (MU)

## Key Insights
•	June 2020 recorded only 15,661 MU — 84% lower than June 2019 (worst COVID impact month)
•	Maharashtra is the highest consuming state (217,080 MU combined)
•	Western region leads nationally with 31.2% share of total consumption
•	NE region has only 1.2% share — highlighting an infrastructure gap
•	Most states recovered only 64–76% of pre-lockdown levels by December 2020
